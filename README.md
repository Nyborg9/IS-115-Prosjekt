# Søknadssystem for Læringsassistenter. __
Skrevet i PHP og kaller til en phpmyadmin database igjennom XAMPP.__
##Det er 3 ulike roller i systemet:__
  **Vanlig bruker:** Kan endre på personlige detaljer, sende inn søknader på jobbutlysninger (Søknadstekst + PDF av en CV) og administrere sine egne jobbsøknader.__
  **Arbeidsgivere:** Kan endre på personlige detaljer, lage jobbutlysninger, administrere egne jobbutlysninger (Gjøre endringer / slette dem) og administrere innsendte søknader (Godta/ Avslå)__
  **Administrator:** Kan administrere jobbutlysninger (Endre / slette), brukere (Endre rolle / slette) og søknader (Godta / Avslå / Slette).
