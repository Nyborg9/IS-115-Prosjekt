# Søknadssystem for Læringsassistenter.   
<h3>Skrevet i PHP og kaller til en phpmyadmin database igjennom XAMPP.</h3>  

## Det er 3 ulike roller i systemet: 
  **Vanlig bruker:** Kan endre på personlige detaljer, sende inn søknader på jobbutlysninger (Søknadstekst + PDF av en CV) og administrere sine egne jobbsøknader.  
  **Arbeidsgivere:** Kan endre på personlige detaljer, lage jobbutlysninger, administrere egne jobbutlysninger (Gjøre endringer / slette dem) og administrere innsendte søknader (Godta/ Avslå)  
  **Administrator:** Kan administrere jobbutlysninger (Endre / slette), brukere (Endre rolle / slette) og søknader (Godta / Avslå / Slette).
