# Søknadssystem for Læringsassistenter. 
Skrevet i PHP og kaller til en phpmyadmin database igjennom XAMPP. 
Det er 3 ulike roller i systemet:
  Vanlig bruker: Kan logge seg inn, endre på personlige detaljer, sende inn søknader på jobbutlysninger (Søknadstekst + pdf av en CV) og administrere sine egne jobbsøknader.
